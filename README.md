# Frame Extractor for Android

**High-resolution video frame extraction made simple and fast.**

Frame Extractor is a utility designed for photographers, developers, and creators who need to pull high-quality still images from video files. Built with **Jetpack Compose**, it provides a fluid, modern interface for browsing videos and extracting frames with millisecond precision.

## ✨ Key Features

- **🎞️ Frame-by-Frame Scrubbing:** Navigate through your videos with extreme precision to find the exact moment you need.
- **📸 Lossless Extraction:** Save frames in their original resolution and quality directly from the video stream.
- **⚡ Performance First:** Uses native Android Media APIs for fast, efficient processing without battery drain.
- **📁 Integrated Video Browser:** Quickly select videos from your device's storage via a sleek, thumbnail-driven interface.
- **🔒 Privacy Guaranteed:** 100% offline. No internet permissions, no tracking, and no data collection.

## 📱 Specifications
- **Package ID:** `com.tailgunnerx.frameextractor`
- **License:** [Apache License 2.0](LICENSE)
- **OS Support:** Android 7.0+ (API 24+)
- **Build System:** Gradle 8.10.2

## 🛠️ Tech Stack
- **Language:** Kotlin
- **UI:** Jetpack Compose (Material 3)
- **Engine:** Android `MediaMetadataRetriever` and `MediaExtractor` for high-fidelity frame capture.

## 🚀 Getting Started

1. **Clone the Repo:** `git clone https://github.com/tailgunnerx/FrameExtractor-Android.git`
2. **Open in Android Studio:** Select the root project folder.
3. **Build & Run:** Deploy to your device and start extracting.
